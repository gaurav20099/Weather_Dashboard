# 🌦️ Weather Dashboard

A modern and responsive Weather Dashboard that allows users to check real-time weather conditions for any city around the world. The application fetches live weather data using a weather API and displays important weather information in a clean and user-friendly interface.

## 🚀 Features

- 🌍 Search weather by city name
- 🌡️ Real-time temperature
- 💧 Humidity information
- 🌬️ Wind speed
- ☁️ Weather conditions with icons
- 📱 Fully responsive design
- ⚡ Fast and easy-to-use interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Weather API (OpenWeatherMap or similar)
